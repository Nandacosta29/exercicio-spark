# 📊 Resolução do Exercício: Spark DataFrames & SQL

Olá! 👋 Este repositório contém a resolução da atividade prática sobre **Apache Spark**, desenvolvida como parte do meu percurso de aprendizagem em Ciência de Dados.

## 📝 Sobre o Projeto
O objetivo principal deste projeto é demonstrar a manipulação de dados em larga escala utilizando o **PySpark**. Através de um conjunto de dados simulando um ecossistema de vendas, realizei análises exploratórias e transformações focadas em:
* Performance de vendas por categoria de produto.
* Identificação e ranking de fidelidade de clientes.

---

## 🚀 Tecnologias e Ferramentas
* **Python 3.x**
* **Apache Spark (PySpark)**
* **Google Colab** (Ambiente de desenvolvimento)
* **GitHub** (Hospedagem de dados e versionamento)

---

## 🔍 Metodologia de Resolução
Para este desafio, apliquei uma abordagem comparativa, resolvendo os problemas de negócio de duas formas distintas:
1. **DataFrame API:** Utilizando métodos nativos do Spark (`.join`, `.groupBy`, `.agg`).
2. **Spark SQL:** Utilizando consultas estruturadas em linguagem SQL padrão após o registro de tabelas temporárias.

> Essa dualidade permite entender a flexibilidade do Spark em diferentes contextos de engenharia e análise de dados.

---
